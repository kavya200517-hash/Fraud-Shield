# 🌱🚀🌾 Grow-Wise 🌾🚀🌱

Grow-Wise is an AI-powered web platform helping farmers & agricultural professionals make smarter decisions.  
Features include crop recommendation, yield prediction, emotion-aware chatbot support & real-time weather insights 🌤️.

---

## 🌟 Features

- Emotion-aware chatbot to assist & guide users  
- Crop recommendation based on soil, climate & region data  
- Yield prediction for upcoming seasons  
- Real-time weather analysis & alerts  
- Modern React-based UI with clean and responsive design  

---

## 🛠️ Tech Stack

- **Frontend:** TypeScript, React, Vite, TailwindCSS  
- **Backend:** Flask (Python)  
- **ML Models:** Scikit-learn (Crop recommendation & yield prediction)  
- **APIs:** Weather API, Gemini API (planned for chatbot)  

---

## ⚡ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/SujaySAK777/grow-wise.git
   cd grow-wise
   ```

2. **Setup Backend (Flask)**
   ```bash
   cd backend
   pip install -r requirements.txt
   python app.py
   ```

3. **Setup Frontend (React + Vite)**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. **Open in Browser**
   - Frontend: `http://localhost:5173`  
   - Backend API: `http://localhost:5000`
